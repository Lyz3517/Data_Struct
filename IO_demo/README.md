# 进程

**1.1进程的理论知识**
（1）进程：程序一次执行过程，是系统资源调度的最小单位，是动态的。
（2）程序：存储在磁盘的指令的有序集合，是静态的。

**1.2 进程的内容**
数据段：全局变量、malloc分配的空间
正文段：程序的代码行
堆栈段：函数返回值、参数、局部变量等。

**1.3 进程的内存管理**
正文段、用户数据段、系统数据段

进程号（PID）：唯一标识的一个进程

**进程的类型**：
（1）交互进程：由shell控制和运行的进程，也就是在终端中运行产生的进程。
ctrl+z：使进程进入挂起状态，被挂起的进程e被称为作业
jobs -l：查看被挂起的进程
bg % 作业号：使这个作业恢复前台运行，不能ctrl+c结束
