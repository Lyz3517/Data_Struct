# 无名管道

**1、定义：**
是一个在内核中存在的特殊文件（看不到文件名），使用文件IO进行数据交互。 

2、特点：
- 一种半双工通信（数据只能一个方向a传递），有固定的读端和写端
- 必须在具有a亲缘关系的进程之间使用。

3、api接口
- 创建无名管道文件
- int pipe(int fd[2]);
- 头文件：#include <unistd.h>
- 参数：fd：数组名，用来保存读端和写端的文件描述符
- 返回值：成功返回0 失败返回-1
- pipefd[2]:无名管道的两个文件描述符，int型的数组，大小为2、
- pipefd[0]为读端，pipefd[1]为写端
